# [프로젝트] React - 상품 주문 사이트 구현
## 프로젝트 소개 😎
상품 주문 사이트입니다. 
본 사이트에서는 고객이 상품을 추가하여 주문할 수 있습니다. 


## 추가 구현 부분
- 감가 기능 추가
- 결제 정보 (email, address, postcode) 유효성 검사
- 결제 완료 후, 고객 기입 칸 리셋하기
- re-rendering 방지 시도 ( React.memo 함수 사용)
